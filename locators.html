Locators Cheat Sheet XPath & CSS — Selenide / Selenium

Схемы, примеры, и варианты кода. В колонке Selenide даны несколько наиболее удобных способов: $("..."), byXpath, Selectors.byAttribute, byText/withText и др.

XPath

Rule Schema Example Comment Selenide (варианты) Selenium
tag & attribute //tag[@attr="value"] //input[@aria-label="Search"] Находит элемент по атрибуту $("input[aria-label='Search']")
$(byAttribute("aria-label","Search"))
$(byXpath("//input[@aria-label='Search']")) 		driver.findElement(By.xpath("//input[@aria-label='Search']"))
text() //tag[text()="value"] //button[text()="Login"] Полное совпадение текста $(byText("Login"))
$$("button").findBy(text("Login"))
$(byXpath("//button[text()='Login']")) 		By.xpath("//button[text()='Login']")
contains() (text) //tag[contains(text(),"value")] //div[contains(text(),"Error")] Частичное совпадение текста $(withText("Error"))
$(byXpath("//div[contains(text(),'Error')]")) 		By.xpath("//div[contains(text(),'Error')]")
contains() (attr) //tag[contains(@attr,"value")] //input[contains(@placeholder,"user")] Частичное совпадение атрибута $("input[placeholder*='user']")
$(byAttribute("placeholder","user")) (содержит)
$(byXpath("//input[contains(@placeholder,'user')]")) 		By.xpath("//input[contains(@placeholder,'user')]")
multiple attributes //tag[@a1="v1"][@a2="v2"] //input[@type="text"][@name="login"] Несколько условий $("input[type='text'][name='login']")
$(byCssSelector("input[type='text'][name='login']"))
$(byXpath("//input[@type='text'][@name='login']")) 		By.xpath("//input[@type='text'][@name='login']")
moving through the tree //parent/child //div/span Переход родитель → потомок $("div span")
$(byXpath("//div/span")) 		By.xpath("//div/span")
not() //tag[not(contains(@attr,"v"))] //input[not(contains(@type,"hidden"))] Исключение по условию $(byXpath("//input[not(contains(@type,'hidden'))]")) By.xpath("//input[not(contains(@type,'hidden'))]")
indexing (//tag[@attr="v"])[N] (//input[@type="text"])[2] N‑й элемент в выборке $$("input[type='text']").get(1)
$(byXpath("(//input[@type='text'])[2]")) 		By.xpath("(//input[@type='text'])[2]")
parent //tag/.. //input/.. Переход к родителю $(byXpath("//input/..")) By.xpath("//input/..")
following-sibling //tag/following-sibling::tag2 //label/following-sibling::input Сосед справа на том же уровне $(byXpath("//label/following-sibling::input")) By.xpath("//label/following-sibling::input")
preceding-sibling //tag/preceding-sibling::tag2 //input/preceding-sibling::label Сосед слева $(byXpath("//input/preceding-sibling::label")) By.xpath("//input/preceding-sibling::label")
ancestor //tag/ancestor::X //span/ancestor::form Любой предок выше по дереву $(byXpath("//span/ancestor::form")) By.xpath("//span/ancestor::form")

CSS Selectors

Rule Schema Example Comment Selenide (варианты) Selenium
tag & attribute tag[attr="value"] input[aria-label="Search"] По атрибуту $("input[aria-label='Search']")
$(byAttribute("aria-label","Search"))
$(byCssSelector("input[aria-label='Search']")) 		By.cssSelector("input[aria-label='Search']")
id #id #loginBtn По id $("#loginBtn")
$(byId("loginBtn")) 		By.id("loginBtn")
class name .className .btn-primary По классу $(".btn-primary")
$(byClassName("btn-primary")) 		By.className("btn-primary")
multiple class names .class1.class2 .btn.large Два класса одновременно $(".btn.large")
$(byCssSelector(".btn.large")) 		By.cssSelector(".btn.large")
attribute *= [attr*="value"] input[placeholder*="user"] Атрибут содержит $("input[placeholder*='user']")
$(byAttribute("placeholder","user")) 		By.cssSelector("input[placeholder*='user']")
attribute ^= [attr^="value"] input[placeholder^="user"] Атрибут начинается с $("input[placeholder^='user']")
$(byCssSelector("input[placeholder^='user']")) 		By.cssSelector("input[placeholder^='user']")
attribute $= [attr$="value"] img[src$=".png"] Атрибут заканчивается на $("img[src$='.png']")
$(byCssSelector("img[src$='.png']")) 		By.cssSelector("img[src$='.png']")
name / link text (полезно знать) a:contains(text) — (нет в чистом CSS) Ссылки по тексту В CSS нет поиска по тексту — используем API $(byLinkText("Home"))
$(byPartialLinkText("Hom")) 		By.linkText("Home")
By.partialLinkText("Hom")
