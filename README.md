# Web Site Automation Documentation

Обучающий сайт с материалами по автоматизации тестирования веб-приложений.

## Описание

Этот сайт содержит практические материалы и примеры кода для разработчиков, работающих с автоматизацией тестирования. Включает в себя:

### Основные разделы

- **Selenoid** - настройка и использование Selenoid для удаленного запуска браузеров
- **Локаторы** - справочник по XPath и CSS селекторам для Selenide/Selenium
- **Зависимости** - список необходимых Java библиотек для автотестов
- **Рецепты кода (Cookbook)** - практические примеры и паттерны:
  - Конфигурации - примеры настройки тестовых окружений
  - Кастомный генератор сущностей
  - Кастомный компаратор сущностей
  - Кастомные JUnit экстеншены
  - Логин через JavaScript в UI-тестах
  - Кастомный логгер
  - AssertJ Maven конфигурация
- **Bookstore App** - документация по тестовому приложению Spring Digital Bookstore
- **Nbank app** - документация по тестовому приложению Nobugs Bank
- **Kafka** - материалы по работе с Kafka в тестах
- **Docker** - примеры docker-compose конфигураций (включая видео)
- **Browsers** - настройка браузеров для тестирования
- **Local App** - инструкции по локальному развертыванию приложений
