Spring Digital Bookstore API

Это REST API для цифрового книжного магазина. Поддерживает управление книгами, отзывами, рейтингами и скачивание PDF файлов. Может использоваться QA для тренировки тестирования бэкенда в связке с БД.

Важно. Данный проект был создан исключительно в целях самообучения при изучении Spring Boot в далеком 2023 году. Может содержать (и умышленно содержит) баги. Активно не поддерживается, однако работает.
